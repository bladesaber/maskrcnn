# MaskRcnn
Reproduce MaskRcnn and testing it in other dataset. more performence of different data set will be upload soon.

require:  
tensorflow 2.0  
imgaug  
keras 2.3.1
skimage 0.14.4

pretrain model:
https://github.com/matterport/Mask_RCNN/releases/download/v2.0/mask_rcnn_coco.h5
